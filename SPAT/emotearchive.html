# emote-archive🔗

Drew Linky

2nd of May 2026

I keep swearing to myself I'll put this up eventually, and I suppose I'm finally getting around to it now that the document is done. This list is what it says on the label: a repository for every emote I'm personally aware of that we used. Certainly missing several examples, and if you happen to find one you know where to find me.

For visual spacing reasons I'm using a CSS class that makes them smaller, but they'll still be the full-sized version if you save them anyway. Just in case, I've made it so if you click on them the full-sized version will pop up independently. For convenience, I have also included a .zip containing everything here and a spreadsheet for stream emotes here which details whether they were for the weekend stream, the Homestuck community stream, or both.

In the course of organizing this, I realized many of the emotes we used on the server got mirrored into the stream emotes. I might go through and reorganize it, but assume there was some intermixing regardless. Also, many of the emotes used during streams were ripped directly from various Twitch streamers like Vinesauce.

This list is organized alphabetically with server and stream emotes separated, with a couple more arbitrary distinctions within. Clones between server and stream emotes are retained for thoroughness. Much thanks to anyone who hunted these down, and especially Nero for the more thorough search. If we discover more, I'll probably update this page.
